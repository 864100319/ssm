## 所需技术

本次期末项目采用以下技术，其中又分为前端部分和后端部分

### 后端部分

1. Spring + SpringMVC + MyBatis
2. Apache POI（处理word，excel）
3. pageHelper（处理分页）

### 前端部分

1. layUI
2. jQuery（暂定）
